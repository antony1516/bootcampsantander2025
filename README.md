 Sobre o Projeto

Esta página reúne os principais tópicos abordados no primeiro módulo do bootcamp, apresentando de forma prática os conceitos essenciais de HTML, desde a estrutura básica até o uso de semântica e mídias.


Conteúdo

Introdução ao HTML na prática

Apresentação da linguagem de marcação HTML, sua estrutura de elementos e tags, e como eles definem o conteúdo de uma página web.

Trabalhando com formulários

Criação de formulários com a tag <form> para coleta de dados dos usuários, incluindo campos de texto, caixas de seleção e botões de rádio.

Estruturando seu HTML + Formatações

Uso de tags de formatação como <b>, <i> e <u>, além da aplicação de CSS para personalizar o estilo da página.

Trabalhando com mídias utilizando o HTML

Incorporação de imagens, áudio e vídeo com as tags <img>, <audio> e <video>, incluindo seus atributos de controle.

Criando tabelas com HTML

Organização de dados em linhas e colunas utilizando <table>, <thead>, <tbody> e <tfoot>.

Entendendo o HTML semântico

Uso de tags semânticas como <header>, <main>, <section> e <footer> para tornar o código mais legível, acessível e otimizado para SEO.
